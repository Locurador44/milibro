Backend

¿Que es un Backend?

Un backend en los sistemas corporativos lo utilizamos para dirigir en una web o empresa sistemas de gestión de pedidos, inventario y procesamiento de suministro. Este sistema recoge información de los usuarios u otros sistemas de tratamiento de datos en la compañía. Es el encargado de gestionar la información que proporciona el usuario, recogida por el sitio web.

Herramientas de Backend

  • Django (Python)
  • Node.js (JavaScript)
  • Laravel (PHP)
  • Ruby on Rails (Ruby)
