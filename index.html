Tecnologías Web

by: Jesus Mera

¿Que son las Tecnologias Web?

Las diferentes tecnologías permiten crear interfaces de usuario y establecer las comunicaciones con el servidor, además de implementar comportamientos de la web en el servidor. Frontend es la parte de un sitio web que interactúa con los usuarios, por eso decimos que está del lado del cliente. Backend es la parte que se conecta con la base de datos y el servidor que utiliza dicho sitio web, por eso decimos que el backend corre del lado del servidor.

Tabla comparativa de ventajas y desventajas
Ventajas
  1. Son muy fáciles de usar y aprender
  2. Facilitan el trabajo colaborativo
  3. Son escalables y actualizables
  4. Es libre, por lo que se presenta como una alternativa de fácil acceso para todos
Desventajas
  1. Se necesita instalar un servidor web
  2. Todo el trabajo lo realiza el servidor y no delega al cliente
  3. La legibilidad del código puede verse afectada al mezclar sentencias HTML y PHP
  4. Dificulta la organización por capas de la aplicación
